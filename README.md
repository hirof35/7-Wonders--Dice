
# 7 Wonders: Dice Game (Node.js Console Mock)

ボードゲーム『世界の七不思議（7 Wonders）』のダイス・アンド・ライト系のシステムにインスパイアされて作成した、Node.js環境で動作するコンソール型の1人プレイ用ダイスゲーム（プロトタイプ）です。

共通のダイスプールからダイスをドラフト（選択）し、自分の都市（七不思議、軍事、科学、商業など）を成長させて最高得点を目指します。

## 🛠️ 機能と特徴

- **ダイスドラフトシステム**: 毎ターンランダムに振られる3つのダイスから1つを選択。
- **プレイヤーボード管理**: 七不思議の建造進捗、軍事力、コイン、科学トークン、勝利点をリアルタイムにトラッキング。
- **ゲーム終了とスコア計算**: 5ターンが経過するか、七不思議が最大（レベル5）まで完成するとゲームが終了し、最終スコアが自動計算されます。

### ダイスの種類（面構成）
- 🏛️ **灰色 (資源)**: 七不思議の建設を進めます。
- ⚔️ **赤色 (軍事)**: 都市の軍事力を高めます。
- 📜 **青色 (勝利点)**: 純粋な建物勝利点を獲得します。
- 💰 **黄色 (商業)**: コインを獲得します（終了時、3コイン＝1点）。
- ⚙️ **緑色 (科学)**: 科学トークン（羅針盤、歯車、日時計）を獲得します。

## 🚀 動作確認・実行方法

### 1. 前提条件
お使いの環境に **Node.js** がインストールされている必要があります。
- Node.js 推奨バージョン: v14以上

### 2. ファイルの準備
任意のディレクトリに `7wonders-dice.js` という名前でスクリプトファイルを保存します。

### 3. ゲームの実行
ターミナル（またはコマンドプロンプト）を開き、ファイルが保存されているディレクトリに移動して以下のコマンドを実行します。

```bash
node 7wonders-dice.js
🎮 遊び方
ゲームが始まると、現在の「あなたの都市の状態」と、場に振られた3つの「ダイスプール」が表示されます。

欲しいダイスの番号（1〜3）をキーボードから入力し、Enterキーを押します。

選択したダイスの効果が都市に適用され、次のターンに進みます。

終了条件（5ターン経過、または七不思議が5段階まで完成）を満たすと、それまでに集めた資産から最終勝利点が算出されます。
